## 🚀 Getting Started

1. Clone the repository:

```bash
https://github.com/branjesusdev/flypass-find-movies.git
```

2. Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
ultra install
```

3. Run the development server:

```bash
npm run start
# or
yarn start
# or
pnpm run start
# or
ultra start
```

Open up [http://localhost:4200](http://localhost:4200) to view the website 🚀.
